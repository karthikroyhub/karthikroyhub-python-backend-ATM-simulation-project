ATM Simulation System (Python)

Project Overview
This project is a command-line based ATM Simulation System developed in Python. It replicates basic banking operations such as user authentication, deposit, withdrawal, and transaction history management. The project is designed to strengthen fundamental programming concepts and logical thinking.

Features

Secure PIN-based login with limited attempts
Deposit functionality with validation
Withdrawal with balance checking
Transaction history tracking during session
Controlled program exit
Input validation using exception handling

Concepts Used

Conditional statements (if-else)
Looping (while)
Exception handling (try-except)
Match-case (Python 3.10 and above)
Lists for storing transaction history
User input handling

How It Works

The user is prompted to enter a PIN with a maximum of three attempts.
After successful authentication, the ATM menu is displayed.
The user can choose to deposit, withdraw, view transaction history, or exit.
All transactions are recorded and displayed within the session.

Tech Stack
